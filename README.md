## 已包含内容

- 个人首屏：头像、身份、GitHub 链接和站内联系入口
- 折叠式信息区：当前关注、能力说明、近期状态
- 文章与项目手风琴布局，减弱大块卡片的压迫感
- Canvas 光团、能量连线、鼠标扰动、点击涟漪等动态背景
- 自定义动态 SVG 头像资产，适配整站的青蓝、洋红、紫色氛围
- 桌面与移动端响应式布局
